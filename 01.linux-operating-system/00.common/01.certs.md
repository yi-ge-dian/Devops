# 1. CA 机构

1. 自建 CA

```shell
openssl genrsa -out ca.key
# 不指定算法不需要输入密码
```

2. 生成 CA 待签名证书

```shell
openssl req -new -key ca.key -out ca.csr
# 输入密码 123456
# CN
# HeBei
# XiongAn
# zongbu
# Operation
# zongbu
# zongbu@test.com
# 空
# 空
```

3. 生成 CA 根证书

```shell
openssl x509 -req -in ca.csr -extensions v3_ca -signkey ca.key -out ca.crt
```

# 2. 服务器证书

1. 生成服务器私钥 key

```shell
openssl genrsa -des3 -out server.key
openssl genrsa -out server.key
# 指定算法需要输入密码
# 输入密码 123456
# 再次输入密码 123456
```

2. 由服务器私钥创建服务器待签名证书

```shell
openssl req -new -key server.key -out server.csr
# 输入密码 123456
# CN
# HeBei
# XiongAn
# zidanwei
# Operation
# zidanwei
# zidanwei@test.com
# 空
# 空
```

3. 利用 CA 对服务器证书签名

```shell
openssl x509 -days 36500 -req -in server.csr -extensions v3_req -CAkey ca.key -CA ca.crt -CAcreateserial -out server.crt
```

# 3. 接入 nginx

把 server.crt 和 server.key 上传到 nginx

```shell
"server": {
    "listen": "443 ssl",
    "server_name": "xx.com",
    "ssl_certificate": "/data/server.crt",
    "ssl_certificate_key": "/data/server.key"
}
```

# 4. 去除不安全认证

去除不安全的情况

1. 浏览器导入 CA 证书，ca.crt
2. 若上一步不行，操作系统导入 CA 证书，ca.crt

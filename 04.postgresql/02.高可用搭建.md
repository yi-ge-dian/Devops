1. 环境规划

repmgr（监控复制过程、自动故障切换）witness（防止脑裂）

主库：10.0.0.61  + repmgr + pg15.5 + master

备库：10.0.0.62  + repmgr + pg15.5 + standby

仲裁：10.0.0.63  + repmgr + witness

2. 准备环境
主库和备考执行 install-pg15.5 的脚本


3. 安装 repmgr

主库执行
```shell
./install-repmgr-x86.sh 10.0.0.61 1 primary
```

从库执行
```shell
./install-repmgr-x86.sh 10.0.0.62 2 standby
```
